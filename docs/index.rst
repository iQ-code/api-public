Inspiration-Q SDK's documentation
=================================

Inspiration-Q SDK is a Python-based programming interface to access Inspiration-Q's quantum inspired solutions, iQ-Xtreme, iQ-Finance and iQ-ML. This documentation describes the installation and use of the Python interface, the problems that can be solved and the different quantum-inspired and classical algorithms that our libraries offer.
